UI Helpers & Logger – Test Cases# UI Helpers & Logger – Test Cases **(責務 ↔ 観点 対応版)**

UI 描画ヘルパ & ロギング層の **「各メソッドの責務 (Responsibility)」** と
それを検証する **「テスト観点 (What/Why を示す)」** を 1 対 1 で対応させました。

---

## 1️⃣ 単体テスト (Unit)

### 1.1 `Cube3DWidget` – 描画/入力ヘルパ群

| ID         | メソッド                     | **責務 (What)**              | **テスト観点 (Why / How)**             | 入力例                | 期待結果                               |
| ---------- | ------------------------ | -------------------------- | --------------------------------- | ------------------ | ---------------------------------- |
| UI-SEL-01  | `_get_cell_at_touch`     | 画面座標を (x,y,z) セル座標へマッピングする | **中央ヒットで正変換**: 代表値でマッピングが正確か      | `touch=(150,220)`  | `(1,1,1)` を返す                      |
| UI-SEL-02  | "                        | 同上                         | **境界外クリックを無視**: 非セル領域では `None` 返却 | `touch=(-10,0)`    | `None`                             |
| UI-HL-01   | `_handle_cell_selection` | セル選択を状態に反映しハイライト描画         | **初回選択で index セット**               | 初回 `(0,0,0)`       | `selected_index==(0,0,0)` & redraw |
| UI-HL-02   | "                        | 同上                         | **既選択セルに再入力 → attempt\_input 呼出** | 連続タップ              | `on_success_input` が1回呼ばれる         |
| UI-DRW-01  | `_draw_cube`             | キューブ全体を Canvas へ描画する       | **正常描画完了**: インストラクションが生成され例外なし    | —                  | Canvas 指示数 > 0                     |
| UI-DRW-02  | `_draw_slice_view`       | スライスビューを描画                 | **軸・深度変更で再描画**                    | 軸 `Z`, depth 2     | Canvas クリア → 再生成                   |
| UI-RW-01   | `redraw`                 | 現在状態を全面再描画                 | **状態遷移後 redraw**                  | board 更新後          | Canvas が最新状態を反映                    |
| UI-INFO-01 | `redraw_info`            | 数列やスコア表示を更新                | **数字更新でラベル同期**                    | `current_number=5` | ラベル text "5"                       |

> 💡 **Why が先に来るように**「責務 → それを担保するテスト観点」順で記述しています。

| メソッド                 | 責務        | 代表テスト観点                           |
| -------------------- | --------- | --------------------------------- |
| `_draw_numbers`      | 各セルに数字を描画 | 任意セルに数字=3 を入れた後、テキスト指示が "3" であること |
| `_draw_grid`         | グリッド線描画   | 軸 XY のセル数に応じて Line 数が変わるか         |
| `_draw_highlight`    | 選択セルハイライト | index=(1,1,1) で矩形座標が一致            |
| `_draw_text_in_cell` | セル内文字描画   | フォントサイズ計算が box\_size 比の ±1px 以内   |
