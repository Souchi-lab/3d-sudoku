## 3D 数独 UI – Kivy 技術設計メモ

### 1. スコープ

> **対象に含める**
>
> * 3D 描画・入出力まわり
> * アプリ構成 / 依存ライブラリ
> * カメラ・タッチ操作実装
>
> **対象に含めない**
>
> * ゲームルール / パズル生成・判定処理
> * データベース設計やビジネスロジック

---

### 2. 技術スタック

| レイヤ           | 採用 Tech                | 補足                         |
| ------------- | ---------------------- | -------------------------- |
| UI / アプリ枠組    | **Kivy** (2.3+)        | クロスプラットフォーム。タッチ／キーボード入力を吸収 |
| 3D 描画         | **moderngl**           | OpenGL 3.3+ バインディング        |
| 行列演算          | **pyrr**               | `mat4`, `vec3` などで視点操作を簡素化 |
| (任意) 3D シーン管理 | `kivy3` or `kivy3dgui` | モデル＆カメラのノード管理が楽になる         |

> **Python 3.10 以上** を前提推奨。
> 必要パッケージ：`pip install kivy moderngl pyrr`

---

### 3. フォルダ構成案

```
sochi_sudoku_3d/
├─ main.py            # Kivy App エントリ
├─ core/
│   ├─ renderer.py    # moderngl 初期化 & 描画ループ
│   ├─ camera.py      # 視点・ズーム制御
│   ├─ cube.py        # 単体キューブのシェーダ/メッシュ
│   └─ board.py       # 3D グリッド配置・ピッキング
├─ ui/
│   ├─ panels.py      # 数字パレット・ボタン等（2D）
│   └─ styles.kv      # Kivy スタイル記述
└─ resources/
    └─ textures/      # 数字付きフェイス画像など
```

---

### 4. 描画パイプライン

1. **moderngl.Context** 生成
2. **Uniform Buffer** にカメラ行列 (view / projection) を送信
3. **Instanced drawing**

   * 同じ `Cube` メッシュを27回描く
   * オフセット行列 & 色を `instance_buffer` に詰める
4. 毎フレーム：`Clock.schedule_interval(renderer.draw, 1/60)`

---

### 5. カメラ & 操作バインド

| 操作       | 実装フック例                                            | 備考                           |
| -------- | ------------------------------------------------- | ---------------------------- |
| 回転（ドラッグ） | `on_touch_move` → `camera.orbit(dx, dy)`          | 感度は画面サイズで正規化                 |
| ズーム（ピンチ） | `on_touch_down/up` の距離変化 → `camera.dolly(factor)` | Wheel 対応なら `on_mouse_scroll` |
| キューブ選択   | **Ray picking**：タップ座標→Ray→各 AABB ヒット判定            | ヒットした `(x,y,z)` をコールバック      |

---

### 6. クラス粒度

```python
# camera.py
class Camera:
    pos     : vec3
    target  : vec3
    fov     : float
    def orbit(self, dx, dy): ...
    def dolly(self, dz): ...

# cube.py
class Cube:
    def __init__(self, position: vec3, color: tuple):
        self.model_mat = mat4.create_from_translation(position)
        self.color = color
```

`renderer.py` で `Cube` 配列をループ → instanced 描画。

---

### 7. UI (2D レイヤ)

* `FloatLayout` を画面最上位に重ね、`Widget` 系で

  * 数字ボタン
  * Undo / Reset
* Kivy KV 言語でスタイルを切り分け
* 3D 選択 ⇆ パネル入力のイベントバスは `EventDispatcher` 経由

---

### 8. ビルド & 配布

| ターゲット           | ツール           | 備考                                                 |
| --------------- | ------------- | -------------------------------------------------- |
| Windows / macOS | `pyinstaller` | 単一 EXE / app バンドル                                  |
| Android         | **Buildozer** | `buildozer.spec` で `moderngl` を `requirements` に追加 |
| iOS (検証用)       | `kivy-ios`    | Xcode プロジェクト生成後、手動で `OpenGLES` リンク                 |

### 9. 今後の拡張ポイント（技術面のみ）

* **フレーム間差分描画**：変更セルのみ再 upload
* **GPU ピッキング**：カラー ID バッファを使えば O(1) 判定
* **Post-processing**：Bloom / Outline シェーダでハイライト
* **分割シーン**：大規模盤面でも Octree で culling
