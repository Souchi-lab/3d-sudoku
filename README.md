# Hyper-Cube Sudoku

3D数独をベースにした、タイムアタック・オンラインランキング機能を備えたパズルゲームです。PythonとKivyで開発されています。

## 現在の機能

*   N (3x3x3から5x5x5) 可変の3Dキューブ表示
*   タイムアタックモード（スコアと時間の調整機能付き）
*   難易度に応じた初期セル配置
*   プレイヤーのターン管理と数字シーケンス生成
*   基本的なCPUプレイヤー
*   ゲームイベントのログ出力（ボード状態、セル配置など）
*   レベル選択とゲーム表示の基本的なUI
